# e-commerce-store




 ## Technology Which I Used
   ***Backend***
   
 - Nodejs
 - Mongodb
 - Express

 ***Frontend***
 

 - React Js
 - Redux

<br />

 ## Step to Start Project
 

 
**For Backend**

 1. `cd e-comerce-backend`
 2. `yarn`
 3. `yarn data:import`
 4. `yarn dev`

**For Frontend**

 1. `cd e-comerce-frontend`
 2. `yarn`
 3. `yarn start`
